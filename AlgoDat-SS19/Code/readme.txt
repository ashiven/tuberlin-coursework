Die Projekte aus den Aufgaben Vorgaben und Vorlesungsbeispielen importiert man am besten so:

File | New | Project → (Next / Next) Project Location: Hier den Ordner mit den Java Dateien auswählen.

Dieser Weg ist besser als File | Open Project, weil dann die eigenen Einstellung übernommen werden. Mit File | Open Project kann es z.B. passieren, dass man erst eine neue JDK auswählen muss.
